Lesson 03

생성형 AI의 메커니즘

LLM 훈련의 3단계

1. 사전 학습 (Pre-training): 인터넷의 방대한 데이터를 스스로 학습하며 일반적인 상식과 언어 구조를 습득합니다.

2. 미세 조정 (SFT): 인간이 작성한 고품질 Q&A 데이터를 통해 지시사항을 이행하는 법을 배웁니다.

3. 정렬 (RLHF): 인간의 피드백을 통해 더 안전하고 유용한 답변을 하도록 강화 학습을 거칩니다.
